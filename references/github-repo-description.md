# WordsCardEink

E-ink word card generator for Raspberry Pi + Waveshare panels. It selects vocabulary from local datasets or OpenAI, enriches with phonetics and multilingual synonyms, renders a stylized card, and serves updates over HTTP for easy remote triggering.

## Suggested GitHub Description (short)
E‑ink word cards with OpenAI‑powered vocabulary, phonetics, and multilingual synonyms for Raspberry Pi + Waveshare.

## Suggested GitHub About (longer)
A Raspberry Pi e‑ink project that curates vocabulary from CSV or OpenAI, adds phonetics and translations, renders themed word cards, and exposes a simple HTTP API to push new words to Waveshare panels.
